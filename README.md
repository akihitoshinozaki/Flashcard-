# Flashcards

A personal vocabulary flashcard app built for fast learning on mobile.

## Features

- Type a word and auto-generate a definition using AI — no manual lookup needed
- Spaced repetition review system based on the Ebbinghaus forgetting curve: cards return at 1 day, 3 days, 1 week, 2 weeks, and 1 month intervals
- Self-grading review: cards you remember get spaced further out, cards you forget reset sooner
- Works on iPhone via Safari — add to home screen for a native app feel
- All data stored locally on your device

## Setup

1. Open the app and tap ⚙ Setup
2. Add a free API key from [console.groq.com](https://console.groq.com) or [openrouter.ai](https://openrouter.ai)
3. Start adding words
